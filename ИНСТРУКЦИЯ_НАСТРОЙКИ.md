# Пошаговая настройка: от установки APK до работы

## Общая схема

```
Телефон (приложение) → перехватывает уведомления → отправляет на ваш сервер
                                                          ↓
Ваш сервер (бот + API) ← получает уведомления и пересылает в Telegram-канал
```

Сначала настраиваете бота и сервер, потом подключаете приложение на телефоне.

---

## Часть 1. Telegram-бот и канал

### 1.1 Создать бота

1. Откройте в Telegram [@BotFather](https://t.me/BotFather).
2. Отправьте команду: `/newbot`.
3. Введите имя бота (например: «Уведомления с телефона»).
4. Введите username бота (должен заканчиваться на `bot`, например: `my_notify_bot`).
5. BotFather пришлёт **токен** вида `123456789:ABCdefGHI...`. **Скопируйте и сохраните** — он понадобится для сервера.

### 1.2 Создать канал для уведомлений

1. В Telegram: меню → **Создать канал** (или **New Channel**).
2. Название любое (например «Уведомления с телефона»).
3. Тип канала можно оставить **Приватный** (только вы) или сделать публичным — как удобнее.
4. Канал создан.

### 1.3 Добавить бота в канал как админа

1. Откройте канал → значок названия канала вверху → **Управление каналом** (или **Administrators**).
2. **Добавить администратора** → найдите вашего бота по username (например `@my_notify_bot`).
3. Выдайте право **Публикация сообщений** (или **Post messages**). Остальные права можно не давать.
4. Сохраните.

### 1.4 Узнать ID канала

Способ 1 (простой):

1. Опубликуйте в канале любое сообщение (например «тест»).
2. Перешлите это сообщение боту [@userinfobot](https://t.me/userinfobot).
3. В ответе будет **Forwarded from chat:** и число вида **-1001234567890** — это ID канала. Сохраните его (с минусом).

Способ 2: добавьте в канал бота [@getidsbot](https://t.me/getidsbot), он тоже покажет ID канала.

## Часть 2. Сервер с ботом (компьютер или VPS)

Сервер нужен, чтобы бот и API работали 24/7. Можно использовать свой ПК или арендовать VPS (например Timeweb, Selectel, зарубежный хостинг).

### 2.1 Установить Python

- На Windows: скачайте Python 3.10+ с [python.org](https://www.python.org/downloads/), при установке отметьте **Add Python to PATH**.
- На Linux: обычно уже есть; если нет: `sudo apt install python3 python3-pip` (или аналог для вашего дистрибутива).

### 2.2 Скопировать и настроить бота

1. На сервере откройте папку проекта (например `avtomatika`).
2. Перейдите в папку бота:
   ```bash
   cd bot-server
   ```
3. Создайте файл настроек из примера:
   - Windows (cmd): `copy .env.example .env`
   - Linux / Mac: `cp .env.example .env`
4. Откройте файл `.env` в блокноте и заполните:
   ```env
   TELEGRAM_BOT_TOKEN=сюда_токен_от_BotFather
   TELEGRAM_CHANNEL_ID=-1001234567890
   ```
   Вместо `-1001234567890` подставьте свой ID канала.  
   Можно оставить `TELEGRAM_CHANNEL_ID=` пустым и задать канал позже в боте командой `/channel -1001234567890`.

5. Если сервер доступен по адресу в интернете (например `https://my-server.ru`), добавьте в `.env`:
   ```env
   API_BASE_URL=https://my-server.ru
   ```
   Тогда в боте при добавлении устройства будет показываться этот URL для ввода в приложении.

### 2.3 Установить зависимости и запустить

В папке `bot-server` выполните:

```bash
pip install -r requirements.txt
python run_all.py
```

Либо в двух окнах терминала:

- Терминал 1: `uvicorn api:app --host 0.0.0.0 --port 8000`
- Терминал 2: `python run_bot.py`

Окно с ботом должно быть открыто постоянно (или сервер должен работать как служба). Напишите боту в Telegram команду `/start` — если ответил, бот работает.

### 2.4 Доступ к API из интернета (если сервер у вас дома)

- Роутер: пробросьте порт **8000** на IP компьютера, где запущен `uvicorn`.
- Внешний адрес: узнайте свой белый IP (например на [2ip.ru](https://2ip.ru)) или настройте домен и DDNS.
- В приложении на телефоне URL будет: `http://ВАШ_IP:8000` или `https://ваш-домен.ru` (если настроили HTTPS на 8000 порт).

Для постоянной работы удобнее арендовать VPS и там запускать бота и API.

## Часть 3. Приложение на телефоне (после установки APK)

### 3.1 Разрешить доступ к уведомлениям

1. Откройте приложение **«Уведомления в Telegram»**.
2. Нажмите **«Включить в настройках»**.
3. В списке найдите это приложение и включите переключатель **«Разрешить доступ к уведомлениям»**.

Без этого шага приложение не сможет перехватывать уведомления.

### 3.2 Получить токен устройства из бота

1. В Telegram откройте вашего бота (тот, для которого вы указали токен в `.env`).
2. Отправьте команду: `/new` (можно с именем: `/new Мой телефон`).
3. Бот пришлёт сообщение с **токеном устройства** (длинная строка) и, если задан, **URL сервера**.
4. Скопируйте токен (удобно нажать на него — он обычно подсвечивается как код).

### 3.3 Ввести в приложении URL и токен

1. В приложении на телефоне в поле **«Сервер (URL бота)»** введите адрес сервера:
   - Если сервер дома: `http://ВАШ_ВНЕШНИЙ_IP:8000` (без слэша в конце).
   - Если VPS/хостинг: `http://IP_или_домен:8000` или `https://ваш-домен.ru`.
2. В поле **«Токен устройства»** вставьте токен из бота.
3. Нажмите **«Сохранить»**.

Если URL или токен неверные, уведомления на сервер не дойдут.

### 3.4 Указать, от каких приложений перехватывать уведомления

В приложении в блоке **«Приложения для перехвата»** нужно добавлять **пакеты** (package name) приложений.

Примеры пакетов:

| Приложение   | Пакет (package name)     |
|-------------|---------------------------|
| WhatsApp    | `com.whatsapp`            |
| Telegram    | `org.telegram.messenger`  |
| VK          | `com.vkontakte.android`   |
| Gmail       | `com.google.android.gm`   |
| СМС         | `com.android.mms` (зависит от прошивки) |

Как добавить:

1. В поле ввода введите пакет (например `com.whatsapp`).
2. Нажмите **«Добавить»**.
3. Повторите для других приложений.

Как узнать пакет любого приложения:

- Установите [App Inspector](https://play.google.com/store/apps/details?id=com.ohussey.appinspector) или аналог — в описании приложения указан package name.
- Либо найдите в интернете по запросу «package name [название приложения]».

### 3.5 Задать канал в боте (если не задали в .env)

Если в `.env` вы не указывали `TELEGRAM_CHANNEL_ID`, сделайте в боте:

```
/channel -1001234567890
```

Подставьте свой ID канала (с минусом). После этого уведомления с телефона начнут приходить в канал.

## Как пользоваться дальше

- **Уведомления** от выбранных приложений автоматически уходят на сервер и публикуются ботом в канал.
- **Добавить ещё один телефон:** в боте выполните `/new`, скопируйте новый токен и введите его в приложении на втором телефоне (URL сервера тот же).
- **Список устройств:** `/devices`.
- **Изменить имя устройства:** `/setname <токен> Новое имя`.
- **Удалить устройство:** `/delete <токен>` — после этого старый токен перестанет работать.
- **Изменить список приложений:** в приложении на телефоне добавьте или удалите пакеты в блоке «Приложения для перехвата».

---

## Проверка, что всё работает

1. Бот отвечает на `/start` и `/new` — сервер и бот запущены.
2. В приложении статус «Доступ к уведомлениям» — **Включено**, URL и токен сохранены, добавлен хотя бы один пакет (например WhatsApp).
3. Отправьте себе в WhatsApp тестовое сообщение или попросите кого-то написать вам — в канал должно прийти уведомление с заголовком и текстом.

Если уведомления не приходят: проверьте URL (доступен ли с телефона), токен (скопирован полностью), канал (`/channel` в боте) и что приложение не «убито» системой (иногда нужно отключить оптимизацию батареи для этого приложения).
